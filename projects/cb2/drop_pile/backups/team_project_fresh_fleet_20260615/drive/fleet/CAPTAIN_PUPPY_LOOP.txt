## Three tiers (pick one)

### A — Files only (easiest, works NOW)
- Both use same paths on Drive
- Cursor word **PUPPY** / **CAPTN** when mtime changes
- **Lag:** 1–3 min · **Effort:** 0 new code · **Brian:** 0

### B — Poll daemons (recommended, ~1 session)
CB2 + puppy64 each run every 30s:
- if `mac_inbox` mtime new → wake agent / log
- if `puppy_outbox` mtime new → CAPTN sees reply

**Effort:** one `fleet_bus_watch.sh` on Drive, copy to both boxes, systemd or cron.
**Lag:** ~30–60s after sync · **Brian:** 0 once running.

### C — WireGuard fast lane (optional later)
Puppy already has **wg0 10.0.0.1**. Mesh radio **:8765** can push SSE.
Drive stays source of truth; WG for **ping awake** only.
**Effort:** few hours · use after B works.
