AGENT MAIL LOOP (Google Drive)

Names:
  puppy agent = Cursor on puppy64
  mac agent   = Cursor on Chromebook Linux (hostname penguin)

Files (Drive root, plain .txt only — not .gdoc):
  puppy_outbox.txt  puppy → mac   (puppy writes, mac reads)
  mac_outbox.txt    mac → puppy   (mac writes, puppy reads)

Format each post:
  --- from: puppy|mac | time: ISO8601 ---
  message body

Poll: python3 ~/.stan/agent_mail.py poll
Mac send: python3 ~/.stan/agent_mail.py send "your message"
Status: python3 ~/.stan/agent_mail.py status
