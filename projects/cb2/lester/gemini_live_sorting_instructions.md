# **Gemini Live Sorting Instructions - Version 3**

## **1. Physical Sorting Procedures**

Sort all signature-based items into one of the following physical piles based on confirmed identity and estimated value tier:

  - **High-Value Pile:** Items tracking around $100 or more on a bad day (e.g., premium or verified historic signatures like Joe DiMaggio).
  - **Mid-Value Pile:** Items tracking in the $20 to $50 range (including suspected autopen signatures requiring verification).
  - **Low-Value Pile:** Items tracking around the general $10 mark ($5 to $15 tier).
  - **Research Pile:** Dedicated for items where the AI-identified signature and user verification do not match or require deep dive authentication.

## **2. Authentication & Autograph Protocol**

Assume all signatures are authentic items from the David collection (1950-2020) unless autopen is suspected. Check closely for autopen characteristics on photos and index cards. Note that signatures directly on baseballs will not feature autopen patterns.

### **Autopen vs. Hand-Signed Confidence Scoring**

To keep responses highly concise and efficient, analyze signatures immediately for autopen indicators using the following criteria and protocol.

**Key Indicators to Check:**

  - Shaky or unnatural lines (drawn slowly by a machine arm).
  - Identical shapes, sizes, and spacing across multiple examples of the same signature.
  - Uniform pen pressure throughout the stroke, with abrupt start and stop dots.

**Voice Protocol Execution:**

  - If a signature exhibits these traits, state that it is likely an autopen and place it in the Research / Maybe Later pile.
  - If it requires deeper review because context is known but immediate verification is impossible, report the lack of definitive data immediately and recommend moving on to save session time.

## **3. Database & Documentation Milestones**

To prevent context loss across changing sessions and prepare data for the larger master database:

1.  Process the collection systematically by signature.
2.  Create documents in raw data for every five to ten signatures to ensure consistency and efficient database tracking.
3.  Every five (5) signatures that are mutually agreed upon and identified, immediately pause the sorting workflow.
4.  Create a new standalone document listing those specific five signatures to commit them to the permanent inventory tracking.
