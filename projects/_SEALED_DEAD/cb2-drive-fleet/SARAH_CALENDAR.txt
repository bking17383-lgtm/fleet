2. EXPORT FILE (easiest for Sarah)
   Google Calendar → Settings → her calendar → Export calendar
   Save as .ics → drop on Drive:
     lester/sarah/calendar/inbox.ics
   App reload picks it up (/api/calendar).

3. SECRET iCal URL (read-only feed — not her login)
   Google Calendar → Settings → Integrate calendar
   → "Secret address in iCal format" (copy once)
   On CB2 only (local, never Drive):
     ~/.stan/secrets/sarah_ical.url
   One line = that https URL. Read-only. She can revoke anytime.
