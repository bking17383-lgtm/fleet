CURSOR MIC — speak in this agent chat
Updated: 2026-06-14

Mic hardware: OK in Linux (tested).
CAPTN cannot flip a config switch — you enable OS + use Cursor UI.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1) CHROME OS (once, ~30 sec)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Settings → Advanced → Developers → Linux development environment
Turn ON:  Microphone  (and Camera if you want EYES later)

Then:  Shut down Linux → turn back on
(Crostini menu or restart from Settings)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2) CURSOR — speak to CAPTN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

In this Agent chat box:

  Hold  Ctrl+M  → speak → release → text appears

Or click the  microphone icon  in the prompt box (if visible).

Known quirk: mic icon may hide until you type one character — Ctrl+M still works.
